# Golden Dandelion

A Minecraft data pack that gives the golden dandelion a camman18-inspired appearance.

## Description

This pack modifies the textures of the golden dandelion block and item to match a custom camman18 style. Perfect for players who want to personalize their Minecraft experience with unique block aesthetics.

## Features

- Custom golden dandelion block texture
- Custom golden dandelion item texture
- Compatible with Minecraft versions supporting pack format 84-88

## Installation

1. Download the pack from this repository
2. Extract the files to your Minecraft data packs folder:
   - Single Player: `.minecraft/saves/<world-name>/datapacks/`
   - Server: `world/datapacks/`
3. Reload the data pack (run `/reload` in-game or restart the server)

## Compatibility

- **Pack Format:** 84-88
- Supports multiple recent Minecraft versions

## Files

- `pack.mcmeta` - Pack metadata and configuration
- `pack.png` - Pack icon displayed in Minecraft
- `assets/minecraft/textures/block/golden_dandelion.png` - Block texture
- `assets/minecraft/textures/item/golden_dandelion.png` - Item texture
